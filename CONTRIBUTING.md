# Contributing to Packigo

Thank you for your interest in contributing to Packigo! We welcome contributions from everyone.

### 1. Reporting Bugs
- Check the [Issues](https://github.com/GSriCharan12/Packigo/issues) tab to see if the bug has already been reported.
- If not, open a new issue with a clear title and description.
- Include steps to reproduce, expected behavior, and screenshots if applicable.

### 2. Suggesting Enhancements
- Open a new issue with the tag `enhancement`.
- Describe your idea clearly and explain why it would be useful.

### 3. Pull Requests
We accept Pull Requests (PRs) for bug fixes and feature enhancements.

1.  **Fork** the repository and clone it locally.
2.  Create a new branch for your feature: `git checkout -b feature/amazing-feature`.
3.  Commit your changes with descriptive messages.
4.  Push to your branch: `git push origin feature/amazing-feature`.
5.  Open a **Pull Request** and describe your changes.
